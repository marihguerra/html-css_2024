ID and Class selectors

The ID attribute allows you to identify a specific element and create custom style settings only for that one element. In the CSS declaration, the given ID must be preceeded with a "#" symbol. You can also put the element's designation in front of the hashtag.

ex: h1#principal{ }

Alternatively, when you want to give two or more elements the same custom style, you can group them using the Class attribute, identified in the CSS declaration by an "." and the chosen group name. Usually, the name expresses the function or characteristic these elements have in common.

ex: .buttons{ }

One element can be identified with two classes or a class and an id simultaneously according to the pecking order: the ID's custom style overlaps the class' style, since it is a group of unique characteristics for that element. Lastly, there is the "span" tag that can even increase the reach of these functions: you can customize the style of a part of a tag's content, like a word or a sentence, by putting the span "tag" around it and adding the id or class attribute.

Pseudo-classes and pseudo-elements

Pseudo-classes are associated with a special state of an element or class, so they are used for customizing the style of that state, like when the element is being hovered, empty or focused. This function works like a momentary class, that's why it's called "pseudo-class". In the CSS declaration, it is characterized by the presence of ":", preceeded with the element's HTML designation and the state the style will apply to.

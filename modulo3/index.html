Background properties

{background-image}

The background-image property sets one or more background images for an element. By default, it is placed at the top-left corner of an element and it will be repeated both vertically and horizontally to cover the element's size. The value can be set by referencing an image file by url() or using color gradients (linear, radial). It is important to have a background-color behind the background-image.

{background-repeat}

Background-repeat determines how the background-image repetition will work: repeat by default, no repeat, repeat only horizontally (repeat-x) or repeat only vertically (repeat-y).

{background-position}

The background-position is a property that positions where a background-image will start to display inside the element. Examples of existing values are bottom left, top center, bottom right or percentages instead like 50% 50% and 25% 75%.

{background-size}

The background-size property specifies the size of background-images. There are different ways you can set it: entering measurable values (px, percentage..) or using keywords such as contain and cover; "contain" will place your image inside the limits of the element without cutting any of it, and "cover" will fully fill the element's dimensions with the background-image and cover some of it if necessary. This property is being exemplified in the background of this page.

{background-attachment}

The background-attachment will determine how the behaves when the page is scrolled. If set to "scroll", the image will scroll with the page; if set to "fixed" it will be static; and if set to "local" the image will scroll with the element's contents. This property is also being exemplified in the background of this page.

shorthand :

color > image > position > repeat > size > attachment

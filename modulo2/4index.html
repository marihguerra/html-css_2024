Box model

Types of boxes

Box-level

  • elements are placed one below the other;
  • always skip a line, starting in a new one;
  • occupy the entire width of the viewport;
  • examples of tags: div, h1-h6, p, table, ul/ol, main, header, footer;

Inline-level

  • elements are placed one on the left side of the other;
  • occupies the amount of space necessary to exibit the content;
  • doesn't skip a line;
  • examples of tags: span, a, img, button, label;

Display property

The display property is used to specify how an element is shown on a web page. Every HTML element has a default display value, and it is mostly block or inline. This CSS property is used to change the default display behavior of an element.

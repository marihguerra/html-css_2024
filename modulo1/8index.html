Using links

External links

Using the A tag you can add and acess external sources in your site : Duck

The target attribute controls where the destination site will be displayed:

  • _blank will create a new tab/ window in which the site will be displayed.
  • _self will open the site on the current tab, replacing the previous frame, the same with _top and _parent.

Ordered and non ordered lists

To create an ordered list, we use the OL tag to delimit the list and a LI tag to identify each item. We can also decide in which number it will start using the start attribute:

  1. Item 3
  2. Item 4
  3. Item 5

or

  1. Item 1
  2. Item 2
  3. Item 3

or

  1. Item 1
  2. Item 2
  3. Item 3

To create a non ordered list, we use the UL tag instead:

  • Item
  • Item
  • Item

or

  • Item
  • Item
  • Item

or

  • Item
  • Item
  • Item

Putting everything together:

  1. Item 1
    1. Item 1.1
    2. Item 1.2
      • Item 1.2.1
  2. Item 2
    1. Item2.1

Definition list

Definition lists are always inside a DL tag (definition list). Each term comes in a DT tag (definition term) and each descripition comes in a DD (definition description).

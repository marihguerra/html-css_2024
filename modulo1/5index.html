Marks

We can also create a marked/highlighted text using the tag MARK and style.

To have the same color on each marked text, you can place the style in the document's head with the mark selector and its property.

Big and small texts

We're creating a big text and a small text in this paragraph. The tag BIG has become obsolete. Since it has no meaning semantically it would belong in CSS.

Deleted text

We can mark a text as deleted to indicate it should be read but not considered.

Inserted text

We can mark a text as inserted to put empashis, underline.

To underline only, with no semantics, we can use the U tag.

Superscript text

To insert things like: 105, x 2+y.
